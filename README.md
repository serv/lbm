# Local Bookmark (lbm)

## Summary

A simple bookmark manager for terminal.

## Installation



## Usage

```
  Usage:
    Add a bookmark
      $ lbm add [directory]
    Return a bookmark
      $ lbm get [id]
    Go to a bookmark
      $ cd ${lbm get [id]}
    Remove a bookmark
      $ lbm rm [id]
    List bookmarks
      $ lbm ls
```
